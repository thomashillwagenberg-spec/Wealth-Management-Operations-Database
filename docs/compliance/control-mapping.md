# Control mapping

> **Important:** This mapping is a design and evidence index. It is not an audit opinion. A repository or Azure deployment does not make an organization ISO/IEC 27001 certified, ISO/IEC 27701 certified, SOC 2 compliant, SEC compliant, or FINRA compliant. Certification and assurance require an operating management system, assigned owners, risk treatment, policies, recurring evidence, independent assessment, and remediation.

ISO/IEC 42001 is not included because this application does not contain a meaningful AI system.
